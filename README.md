# Jungle
\
这是一 CS109-23S 课程 project 的示例（得分110/100），最终效果可供参考，但注意，**绝对不可以抄袭代码！** \
pdf文件为project要求 \
\
使用方法： \
1、点击右上的绿色的 <> Code 按钮 \
2、点击 Download ZIP \
3、下载完毕后解压 \
4、解压完成后打开，点击Jungle.exe即可运行（注：需要Java13及以上的版本） \
